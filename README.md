# Yaesu FT-8XX Suite by K3LH

Amateur radio control and digital modes application for Yaesu FT-817, FT-818, FT-857 and FT-897 radios.

---

## Installation

1. Download **YaesuFT8XXSuite_Setup_v2.1.0.exe** from above
2. Run the installer and follow the on-screen prompts
3. Launch the app from the desktop shortcut or Start Menu

That's it — no Python or extra tools required.

---

## Getting Started

Once the app is open, click the **Help** menu and select **Help Guide**. Everything you need to get up and running is covered there, including:

- Connecting your radio via CAT (serial port)
- Setting up your audio devices
- Using the voice recorder to record and transmit clips
- Running FT8 / FT4 digital modes via WSJT-X
- Logging QSOs and exporting to ADIF
- DX and POTA spotting

---

## Requirements

- Windows 10 or 11
- A Yaesu FT-817, FT-818, FT-857 or FT-897 radio
- A USB-to-serial CAT cable for your radio
- WSJT-X installed separately if you want digital modes
  - Download free from: https://wsjt.sourceforge.io

---

## Serial Port Access

If the app cannot open your CAT port, try running it as Administrator or check that your USB serial driver is installed correctly.
